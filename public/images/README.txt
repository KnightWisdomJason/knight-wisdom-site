Put your background images here.

Current project uses built-in SVG placeholders:
1) tavern-entrance-bg.svg   (homepage, tavern door outside)
2) tavern-hall-bg.svg       (overall tavern hall page background)
3) tavern-interior-bg.svg   (inside scene panel background)

If you want cinematic visuals, replace them with JPG/WebP files and update
src/app/globals.css URLs accordingly.

Recommended size for photo backgrounds: 1920x1080 or 2560x1440.
